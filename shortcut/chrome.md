| 说明                           | 操作                                  |
| ------------------------------ | ------------------------------------- |
| 打开 Chrome 菜单               | **Alt + F** 或 **Alt + E**            |
| 显示或隐藏书签栏               | **Ctrl + Shift + B**                  |
| 打开书签管理器                 | **Ctrl + Shift + O**                  |
| 在新标签页中打开 “历史记录” 页 | **Ctrl + H**                          |
| 在新标签页中打开 “下载内容” 页 | **Ctrl + J**                          |
| 从页面中的任意位置使用谷歌搜索 | **Ctrl + K** 或 **Ctrl + E**          |
| 跳转到地址栏                   | **Ctrl + l** 或 **Alt + D** 或 **F6** |
